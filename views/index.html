Bienvenido a la App "Gestor de Personas" con Firebase

¿Qué hace esta app?

Esta aplicación permite gestionar una lista de personas, donde puedes:

  • Ver la lista de personas registradas.
  • Insertar nuevas personas con su nombre y teléfono.
  • Editar la información de una persona existente.
  • Eliminar personas de la lista.

¿Cómo funciona Firebase?

Firebase es una plataforma de desarrollo de aplicaciones que proporciona una base de datos en tiempo real. En esta app, Firebase se utiliza para:

  • Almacenar y recuperar datos de personas.
  • Sincronizar datos en tiempo real entre la aplicación y la base de datos.
  • Manejar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) de manera eficiente.
